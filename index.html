Chọn hệ máy

Chọn platform bạn muốn chơi.

NES
Danh sách game từ Google Sheet + hỗ trợ Save/Load state.
SNES
Chạy SNES với core snes9x. Lọc theo cột Platform(s) nếu có.
GBA
Chạy GBA với core mgba. Lọc theo cột Platform(s) nếu có.
PSX (PS1)
Chạy PS1 với core pcsx_rearmed. Cần BIOS cho nhiều game.
NDS
Chạy Nintendo DS bằng EmulatorJS. Cần ROM và thường cần BIOS/firmware (chọn từ máy).
Windows 95 (WASM)
Chạy Windows 95 trong browser bằng js-dos v8 (DOSBox-X). Dùng bundle .jsdos và chạy .EXE bên trong Win95.
