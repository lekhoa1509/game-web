# BIOS (PSX)

Trang PSX sẽ cố load BIOS mặc định từ:

- `bios/SCPH1001.bin`

Lưu ý quan trọng:

- File BIOS PS1 là nội dung có bản quyền. Repo/project này không kèm sẵn BIOS.
- Bạn cần tự cung cấp BIOS từ máy/thiết bị của bạn và đặt đúng tên file như trên.

Gợi ý:

- Nếu bạn dùng BIOS khác, hãy đổi code trong `psx-app.js` (hàm `getDefaultBios()`).
